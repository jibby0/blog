# A simple Astro blog

Powered by [Astro](https://astro.build/) and the [MultiTerm Astro](https://github.com/satnaing/astro-paper) theme.

Live at [jibby.org](https://jibby.org), unless it isn't.

# Usage

```sh
bun install
bun dev
```

Cool things you can do with MultiTerm Astro: https://multiterm.stelclementine.com/posts/showing-off-blog-features/
